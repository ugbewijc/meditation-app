#### Wireframe For Login/Signup Screen
These screens help users to register with their details or log in using their credentials.

    The wireframes should address given user stories:

- As a user, I want to sign up by providing username, email, and password to create an account for the meditation app.
- As a user, I want to log in with my email and password to access my account and perform meditation exercises.
- As a user, I want to see a message if I enter an incorrect email or password, so I know my login credentials are wrong.


<i>Instructions </i>

- Add a logo image for the app of your choice that resembles the meditation.
- Design the layout for the Login page. It should have placeholders for email and password. Also add one button named Login.
- Also, add a SignUp button for users who don't have an account yet.
- Design the layout for the registration form, including fields for name, username, age, and country, along with a button to register.

#### Wireframe For Home Screen
The layout provides users with meditation recommendations and a personalized greeting.

    The wireframe should incorporate the following user stories:

- As a user, I want to see a personalized welcome message with my name on the homepage so that I feel recognized and can confirm I am logged into the correct account.
- As a user, I want to see a section for Popular Meditations with key details for quick exploration.
- As a user, I want to see a Daily Meditation section with a scrollable list of meditation recommendations.

<i>Instructions</i>

- Add a header section at the top of the screen:
    - Include a small logo on the top-left.
    - Add a settings icon on the top-right for settings.

- Insert a personalized welcome message below the header, e.g  "Hello [name]! Find your perfect meditation."
- Create a section for Popular Meditations:
    - Include a rectangular card with a rounded corner for each meditation.
    - Add an image, a tag such as calmness, the meditation title, a brief description, and duration such as 10 minutes.
    - Ensure the card stands out with soft shadows or a light background.

#### Wireframe For Meditation Details Screen
Design a page for a Meditation Details Page that provides a focused view of a specific meditation. The design includes details, options for user interaction, and action buttons.

    The wireframe should incorporate the following UX-related user stories:

- As a user, I want to see detailed information about the meditation so that I can decide if it fits my needs.
- As a user, I want navigation buttons to view About or Instructions for the meditation for clarity on the session.
- As a user, I want action buttons to Add to Favorites to save the meditation for future use.

<i>Instructions</i>

- Add a Header Section:
    - Place a small logo in the top-left corner and add a share icon in the top-right corner.

- Insert the Meditation Image:
    - Add a large rounded rectangle at the top for the meditation image.
    - The image should occupy significant space, drawing the user's attention.

- Include the Meditation Title and Tagline:
    - Place the meditation name such as Mindful Breathing below the image in bold.
    - Add tags like the type of meditation such as calmness and duration such as 10 minutes next to the title in smaller, lighter text.

- Add Navigation Tabs:
    - Include two tabs labeled About and Instructions.
    - Highlight the active tab with a different color or background.

- About Section:
    - Add a section below the tabs with the heading "About [Meditation Name]".
    - Write a short description about the meditation such as Focus on your breath and maintain a steady rhythm to clear your mind and reduce stress.

- Action Buttons:
    - Add a heart icon or like button in the bottom-left for users to express preference.
    - Place a bold button in orange or another accent color labeled Add to Favorites in the bottom center.

- Styling Guidelines:
    - Use minimalist typography with proper hierarchy. Maintain consistent spacing between elements.
    - Use soft colors and rounded corners to enhance a relaxing and meditative theme.

#### Wireframe For Reminder Page Screen
Design a page for a Reminder Page that allows users to set reminders by selecting a date and entering a time. The page also displays all existing reminders for the user.

    The page should incorporate the following UX-related user stories:

- As a user, I want to easily select a date from a calendar so that I can choose the day for my reminder.
- As a user, I want to input a time for the reminder in a clear and intuitive format.
- As a user, I want to see the selected date and time displayed before adding the reminder.
- As a user, I want to view a list of all my added reminders for reference.

<i>Instructions</i>


- Add a Header Section:
    - Place a small app logo in the top-left corner.
    - Add a gear icon in the top-right corner to represent settings.

- Calendar Section:
    - Add a calendar component at the top of the page.
    - Ensure it displays the current month and provides navigation arrows to move between months.

- Time Input Section:
    - Add a text input field below the calendar with a placeholder like Enter Time (HH-MM ).
    - Use a border with rounded corners for the input box.

- Selected Date and Time Display:
    - Add labels for Date such as Date: None and Time such as Time: None, updated dynamically.

- Add Reminder Button:
    - Add a button labeled Add Reminder with a contrasting color and rounded corners.

- Reminders List:
    - Add a section titled All Reminders that displays "No reminders yet" or added reminders in a list.

- Styling Guidelines:
    - Use soft colors, rounded corners, readable fonts, and consistent spacing for a clean, modern design.
